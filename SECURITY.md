# Security Policy

## Reporting

Report security issues privately.

Submit reports through [GitHub Private Vulnerability Reporting](https://github.com/henrydontbbai/henry-image/security/advisories/new). Please do not file public issues or include live credentials, personal data, or exploit details in public discussion.

Include the affected version or commit, reproduction steps or a proof of concept, the security impact, and any known mitigation.

We aim to acknowledge reports within 5 business days and provide a private triage/status update. Please allow reasonable time for coordinated remediation before public disclosure.

## Supported Versions

Only the latest published `1.0.x` patch version is supported for security fixes.
