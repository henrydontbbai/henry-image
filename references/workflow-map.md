# Workflow Map

| Need | Mode |
| --- | --- |
| New image | `generate` |
| Edit existing image | `edit` |
| Many tasks | `batch` |
| Prompt output only | `prompt` |
| Readiness check | `probe` |
| Exact-dimension work | SVG, PDF, or written spec |

Normal path:

1. setup
2. route selection
3. run command
4. review
5. retry or recover
